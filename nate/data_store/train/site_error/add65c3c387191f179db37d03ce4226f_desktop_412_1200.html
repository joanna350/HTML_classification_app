Important notice

The skis are delivered with the mounted but not yet set binding and are not yet ready to be used for winter sports. Before the first use, the binding must be set by a professional or by a specialist sports shop assistant. The ski must be set according to the age, gender, weight, ski boot size and experience level of the skier. The skier runs a great risk of injury if the ski is not set and checked with a ski service machine. Please click here to confirm that you have read and understood this safety notice.
We would also like to inform you that we cannot accept the return of any skis that have been personalised, refined or altered through binding. Any products ordered in conjunction with the ski (such as winter clothing) may, of course, be returned.
